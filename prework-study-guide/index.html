HTML

  • The head element contains info about the webpage.
  • The body element represents the visible conten t shown to the user.
  • The "a" element adds an "anchor" creating a link to the current webpage or to another page, used in junction with the "href" command.
  • At the top of your document the "DOCTYPE (...)" function declares what language the code will be using. Followed in the next line by a "doc tag" which is the root of the code.

Git

  • git status: checks the current branch we are on
  • git checkout (branch name) allows you to switch the branch you are currently working on.
  • git checkout -b (branch name) creates a new branch and switches to it
  • git pull ...
  • git push ...
  • git commit -m is used to commit file changes to your repository and the -m allows you to add a short descriptive message illustrating what changes you made.
  • git clone (paste file) copies a file from Github and brings it to your directory to add to your local filebase.
  • git add -A commits all files in the working branch.
  • git reset will undo all changes to the current git commit including the actual working files
